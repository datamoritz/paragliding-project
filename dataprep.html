1. Paragliding Flight Data

Paragliding flight data were obtained from the Paragliding Forum Leonardo XC server , which hosts one of the largest public repositories of GPS flight tracks worldwide. Each record includes a downloadable .kmz file with the full GPS trajectory isnide, as well as metadata such as glider type, takeoff location, flight date, distance, and duration.

Access was automated with a Python script that logged into the platform using session cookies. The script iterated through Austrian flights of year 2024 only, first downloading the metadata from the flight tables. The metadata was processed and stored in a .csv file for further analysis. This file was used to fetch the corresponding .kmz files in a second scraping step.
