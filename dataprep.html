Gathering of Paragliding Flight Data

Paragliding flight data were obtained from the Paragliding Forum Leonardo XC server , which hosts one of the largest public repositories of GPS flight tracks worldwide. Each record includes a downloadable .kml file with the full GPS trajectory, as well as metadata such as glider type, takeoff location, flight date, distance, and duration.

Access was automated with a Python script that logged into the platform using session cookies. The script iterated through Austrian flights only, downloading both the GPS track (.kml) and metadata from the flight tables. The data were stored in two formats: (1) a folder of raw .kml files (one per flight) and (2) a single metadata.csv file containing structured attributes.
